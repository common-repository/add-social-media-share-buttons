Add Social Media Share Buttons WordPress plugin, adds social buttons via shortcodes, letting users share content on Facebook, X and WhatsApp.

== Description ==
Add Social Media Share Buttons is a FREE WordPress plugin that adds social media share buttons using shortcodes. These buttons let users share statuses, quotes, etc., on social networks like Facebook, X (Twitter), or WhatsApp.


== Installation ==
1. Upload the `add-social-media-share-buttons` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==
Use the shortcode `[asmsb]Your quote here[/asmsb]` to display a quote with social media sharing buttons.
